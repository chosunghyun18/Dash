# Dash — UX 흐름 및 기능 문서

## 화면 구조 개요

```
홈 (친구 친구 목록)
 └── 지인 친구 리스트  ← 친구 클릭 or "소개 보기" 버튼
      └── 나를 소개합니다  ← 지인 클릭
           └── (유료) 지인의 지인 리스트 → 나를 소개합니다
```

---

## 1. 홈 화면 — 친구 목록

내가 추가한 친구들의 목록을 보여주는 메인 화면.

### 레이아웃

- **상단 ~ 하단 탭바 사이**: 친구 리스트 (스크롤 가능)
- **하단 탭바**: 홈 버튼, 마이페이지 버튼 (총 2개)

### 친구 리스트 아이템 구성

| 요소 | 설명 |
|------|------|
| 프로필 사진 | 친구의 대표 이미지 |
| 닉네임 | 친구 이름/닉네임 |
| 프로필 보기 버튼 | → 해당 친구의 프로필 상세 조회 |
| 지인 목록 버튼 | → 지인 친구 리스트로 이동 |

> 리스트 아이템 자체를 클릭해도 "소개 보기"와 동일하게 지인 친구 리스트로 이동

---

## 2. 지인 친구 리스트

친구가 소개해줄 수 있는 사람들의 목록 화면.  
(친구 클릭 or "소개 보기" 버튼으로 진입)

### 접근 경로

- 홈 → 친구 아이템 클릭
- 홈 → 친구 아이템의 "소개 보기" 버튼 클릭
- 홈 → 친구 아이템의 "지인 목록" 버튼 클릭

### 플랜별 노출 범위

| 플랜 | 탐색 깊이 | 설명 |
|------|-----------|------|
| 프로토타입 (무료) | 2단계 (친구의 친구) | 내 친구가 소개해주는 인원까지 |
| 유료 | 3단계 (친구의 친구의 친구) | 각 지인 아이템에 "소개 보기 버튼" 추가 노출 |

### 리스트 아이템 구성

- 지인의 프로필 사진, 닉네임
- 해당 인원 클릭 → **나를 소개합니다** 페이지로 이동
- (유료) "소개 보기 버튼" → 해당 인원의 지인 리스트로 한 단계 더 진입

---

## 3. 나를 소개합니다 (프로필 조회)

다른 사람이 등록한 자기소개 페이지.

### 레이아웃

| 영역 | 내용 |
|------|------|
| 상단 | 프로필 이미지 |
| 중단 | 자기소개글 |
| 하단 | 연락 요청 버튼 |

### 연락 요청 흐름

1. **연락 요청 버튼** 클릭
2. 상대방에게 **푸시 알림** + **데이트 요청** 전송
3. 요청 내역은 **본인 마이페이지**에서 확인 가능

---

## 4. 마이페이지

- 하단 탭바의 "마이페이지" 버튼으로 접근
- 본인 프로필 ("나를 소개합니다") 등록/수정
- 받은 연락 요청 목록 확인
- 보낸 연락 요청 목록 확인

---

## 탐색 깊이 정책 요약

```
나 → 친구 (1단계, 홈에서 관리)
나 → 친구 → 친구의 지인 (2단계, 무료/유료 공통)
나 → 친구 → 친구의 지인 → 지인의 지인 (3단계, 유료 전용)
```
